# Mintlify Clone

A lightweight clone of [Mintlify](https://www.mintlify.com/) - a modern documentation platform designed for developers.

## Overview

This project recreates the landing page design and layout of the original Mintlify website, showcasing key features and sections including navigation, hero section, feature highlights, pricing information, and footer.

## Project Structure

```
├── index.html          # Main HTML file
├── style.css           # Styling and layout
├── assets/             # Images and media files
└── README.md           # This file
```

## Features

- Responsive hero section with call-to-action
- Feature highlight sections
- Enterprise features showcase
- Pricing section
- Footer with links and resources
- Modern glass-morphism design elements

## Getting Started

1. Open `index.html` in your web browser to view the page
2. All styles are defined in `style.css`
3. Images and assets are located in the `assets/` folder
